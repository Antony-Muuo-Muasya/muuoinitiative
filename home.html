Redefining Conflict

Contrary to many pop-psychology answers that our culture gives today,
there are actual conflict resolving techniques that we can use and live by - God's word ... If you are around anyone long enough, or develop friendships of any kind, conflict is bound to happen. Conflict happens each and every day. This is especially true within a marriage and a family. If a family doesn't fight, then that probably means they don't talk to each other. You're a fallen, sinful human. You're married to a fallen, sinful human. You produced fallen, sinful little humans. Conflict is going to happen. So what we need to learn and study is God's word for that is how we go about resolving that conflict Biblically and how to start avoiding those conflicts in the future
